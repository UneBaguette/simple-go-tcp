# Simple Go TCP - **Client** And **Server**

Learning **project** created in May 2025

## Overview

This repository contains a **basic TCP client** and **server** implementation in **Go** for **educational** purposes.

## Features

- Simple TCP server and client
- Request/response communication

## Getting Started

### Prerequisites

- Go 1.21 or higher

### Usage

```bash
# Run the server
go run ./cmd/server

# Run the client
go run ./cmd/client
```
